## ✨ About

This repository contains a **complete TypeScript practice project**, designed for learning and experimenting with **TypeScript**, **HTML**, **CSS**, and project structuring concepts.  

It includes:

- Structured **folders** for `src`, `types`, `utils`, and `dist`.
- TypeScript files with **utility functions** and **type definitions**.
- A `tsconfig.json` for compiling TypeScript.
- A ready-to-use project for practicing **modern web development**.
